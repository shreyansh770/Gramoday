## Gramoday

## HOW TO GET STARTED
1) npm install
2) npm start or nodemon server.js

## TO TEST
1) npm test

## ADDING A REPORT
# POST REQUEST
1) req on http://localhost:8081/reports/
2) add the report

# GET REQUEST
